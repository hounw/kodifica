# Kodifica

Kit de prompt y plantillas para definir proyectos Laravel con docs vivos para agentes.

Usar `kodifica-prompt.md` como prompt inicial.

## Incluye

- `product-definition.md`: que construir.
- `technical-definition.md`: como funciona.
- `project-journal.md`: que cambio y por que.
- `production-runbook.md`: deploy cPanel SSH.
- `github-private-repo.md`: repo GitHub y deploy keys.
- `ai/api-guide.md`: solo si hay API.
